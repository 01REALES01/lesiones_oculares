# Diseño del proyecto

Esta carpeta contiene una **subcarpeta por cada tema** estudiado en el proyecto. En cada una hay un **README.md** con:

- Explicación del tema  
- Enlaces relevantes  
- Códigos de prueba (si aplica)

Los temas están alineados con el **objetivo** y **alcance** del proyecto (plataforma de análisis de retinografías con selección de tres modelos, trazabilidad, métricas e interfaz de apoyo clínico/educativo). Ver el **README principal** del repositorio para objetivo, alcance y riesgos.

| Subcarpeta | Tema |
|------------|------|
| 01-fotografia-fondo-ojo | Fotografía de fondo de ojo (fundus) |
| 02-preprocesamiento-retina | Preprocesamiento (canal verde, CLAHE, Ben Graham) |
| 03-segmentacion-disco-copa-cdr | Segmentación disco/copa y CDR |
| 04-glaucoma-y-retinopatia | Glaucoma y retinopatía diabética |
| 05-modelos-ia-segmentacion-clasificacion | Modelos de IA (segmentación, clasificación, detección) |
| 06-explicabilidad-xai | Explicabilidad (XAI, heatmaps) |
| 07-stack-tecnologico-api | Stack tecnológico y API |
